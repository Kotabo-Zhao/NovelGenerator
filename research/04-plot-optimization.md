# 情节优化与叙事技巧 — 开源项目调研

## 调研项目

### 1. snowflake-novel-subagents（雪花写作法）
- **10 Subagent 覆盖全流程**: 概念设计→三幕架构→人物故事线→概要→人物档案→大纲扩展→人物宝典→场景规划→场景设计→正文写作
- **5 辅助专家**: 类型专家/对话大师/节奏控制师/一致性检查师/编辑审阅师
- **三幕式结构**: 第一幕(建置) → 第二幕(对抗) → 第三幕(解决)

### 2. harnessNovel（拆书+仿写）
- **核心玩法+长短线**: 替代传统大纲，长线持续悬念，舞台内短线阶段性爽点
- **舞台驱动**: stage_roadmap 承担分卷卷纲，story-arcs 基于参考小说叙事模式
- **系统文机制层**: 结构化规则约束数值/状态

### 3. StoryScope 论文（马里兰+DeepMind）
- **5 个 AI 叙事短板**: 爱说教(77%点明主题) / 永远线性(79%无支线) / 身体描写上瘾(81%) / 不打破第四面墙 / 故事撞脸
- **仅用叙事特征区分 AI/人类**: 93.2% 准确率
- **关键启示**: 改词改句式是表面功夫，叙事结构才是根本差异

### 4. PlotCraft
- **三步结构→12章展开→每章12节拍**: 从三幕到章节到节拍的逐级展开
- **参考模式板**: 引用模式→组织角色/地点/事件→生成章节

## 已落地的叙事机制

1. **三态情感弧线**: 压抑→爆发→余韵（已集成到 Writer prompt）
2. **场景导演三模式**: 动作/情感/对话（已集成）
3. **冲突控制符**: [IN内心/IR人际/EN环境/DE宿命]:强度1-5（已集成）
4. **叙事时间线**: 倒叙/插叙/补叙（已集成）
5. **POV 规则**: 场景边界切换/不偷渡信息/单一场景单POV（已集成）
6. **紧张-放松法则**: 高潮后必有缓冲（已集成）
